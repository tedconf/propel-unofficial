Dodatek C - Używanie z klas MapBuilder

Kiedy Propel generuje obiekty dla twojego projektu, generuje także klasy MapBuilder'a, które są "statycznymi" klasami reprezentującymi strukturę twojej bazy danych. Te klasy są używane do opisania np. które kolumny są podstawowymi kluczami (Primary Keys), kluczami obcymi (Foreign Keys) itd. bez konieczności wykonywania żadnych operacji na bazie danych, które odczytają ich własności.

Mimo że nie musisz wiedzieć niczego o tych klasach – ponieważ są używane wewnątrz podstawowych wygenerowanych klas Peer i głównych klas Propela – pozwalają one na szybki i dogłębny dostęp do metadanych twojej bazy.

require_once("propel/Propel.php");

Propel::init("bookstore-conf.php");

require_once("Book.php");
require_once("Author.php");
require_once("Publisher.php");

require_once("propel/map/DatabaseMap.php");


try {

  $dbMap = Propel::getDatabaseMap("bookstore");

  $tables = $dbMap->getTables();

  foreach ($tables as $tableName => $table) {
    print "Table: $tableName\n";
    $columns = $table->getColumns();
    foreach ($columns as $column) {
      $creoleType = CreoleTypes::getCreoleName($column->getCreoleType());
      print "\tColumn: " . $column->getColumnName() . "\n";
      print "\t\tFQN: " . $column->getFullyQualifiedName() .  "\n";
      if ($column->isPrimaryKey()) {
        print "\t\tPrimary Key: true\n";
      }
      print "\t\tSize: " . $column->getSize() . "\n";
      print "\t\tType: " . $column->getType() . "\n";
      print "\t\tCreole Type: " . $creoleType . "\n";
    }
  }


} catch (Exception $e) {
  die("Exception: " . $e);
}
