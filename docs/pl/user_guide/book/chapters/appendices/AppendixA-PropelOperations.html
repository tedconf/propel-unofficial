Dodatek A - Operacje Propela

Propel korzysta z zadań Phing'a do wspierania operacji na bazie danych i dokumentacji. Nie wszystkie z nich sa dołączone do aktualnej(stabilnej) wersji. Zawsze możesz użyć Phing'a aby obejrzeć listę zadań oraz opisy: 

$> phing -f build-propel.xml -list

Nastepna tabela pokazuje zadania oraz ich aktualny status.

Zadanie Opis Zakończone?
convert-props przekształca plik 'properties' do tablicy PHP Tak
create-db tworzy bazę danych Tak
datadtd generuje 'DATA DTD' dla twojego projektu Tak
datadump zrzuca dane z bazy danych do pliku xml Tak
datasql generuje zapytania sql z danych xml Tak
insert-sql wykonuje wygenerowane zapytanie Tak
om generuje obiekty Peer dla twojego projektu Tak

Każde z tych zadań może byc przywołane poprzez sprecyzowanie zadania podczas uruchamiania phing'a:

$> phing -f build-propel.xml targetname
